# Memoria Técnica - TFG DAM
**Proyecto:** Sistema de Gestión para Restaurante (50 mesas)

## 1. Introducción
Aplicación nativa Android para la gestión de pedidos, categorías y mesas en tiempo real.

## 2. Tecnologías
- **Frontend:** Android Studio (Java)
- **Backend:** API REST (PHP o Node.js) - *Pendiente de definir*
- **Base de Datos:** MySQL
- **Arquitectura:** MVVM (Model-View-ViewModel)
